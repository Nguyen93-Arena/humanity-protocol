# Book Zero

> *The philosophical foundation of Humanity Protocol.*

Book Zero is the Genesis document of Humanity Protocol.

It explains why Humanity Protocol exists, the problems it solves, and the vision for a trusted Human Economy in the AI Era.

This is not a technical specification.

It is the philosophy behind the protocol.
