# Contribution Principles

Before contributing, ask yourself:

- Does this improve trust?
- Does this create value?
- Does this respect privacy?
- Does this benefit real humans?

If the answer is "yes", we welcome your contribution.

---

# Pull Request Process

1. Fork the repository.
2. Create a new branch.
3. Make focused changes.
4. Write clear commit messages.
5. Submit a Pull Request.
6. Participate respectfully in review discussions.
